# 🧠 Genius-Idiot Quiz

Консольное и WinForms приложение с викториной, определяющей "уровень интеллекта" пользователя на основе правильных ответов.

## 🚀 Возможности

- 🎮 Прохождение викторины с ограничением по времени
- ⏱ Таймер (15 секунд на вопрос)
- 🧾 Сохранение результатов в JSON
- 📊 Просмотр истории игр
- ➕ Добавление новых вопросов
- ❌ Удаление пользовательских вопросов
- 🖥 Две версии интерфейса:
  - Console
  - WinForms

## 🏗 Архитектура

Проект разделен на несколько частей:

- **ClassLibrary** — бизнес-логика
  - Работа с вопросами
  - Валидация
  - Логика подсчета результата
- **ConsoleApp** — консольный интерфейс
- **WinFormsApp** — графический интерфейс

## ⚙️ Технологии

- C#
- .NET
- WinForms
- JSON (Newtonsoft.Json)

## 💡 Особенности реализации

- Реализован таймер с обновлением строки в консоли
- Валидация пользовательского ввода (ФИО, вопросы)
- Нормализация текста
- Рандомизация вопросов
- Разделение логики и UI

## 📂 Хранение данных

- Вопросы: `QuestionList.json`
- История игр: `QuizHistory.json`

## ▶️ Запуск проекта

1. Клонировать репозиторий
2. Открыть решение в Visual Studio
3. Выбрать нужный проект:
   - ConsoleApp
   - WinFormsApp
4. Нажать Run
