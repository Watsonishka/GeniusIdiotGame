# 🧠 Genius-Idiot Quiz

Это шуточное консольное и WinForms приложение с викториной, определяющей "уровень интеллекта" пользователя на основе правильных ответов. Проект реализован в двух версиях интерфейса: в консоли и в WinForms. 

## 🏗 Архитектура

Проект разделен на несколько частей:

- **ClassLibrary** — бизнес-логика

  ![кл](https://github.com/user-attachments/assets/7535a215-858f-4b04-a25a-18054af1e874)

- **ConsoleApp** — консольный интерфейс

  ![консоль](https://github.com/user-attachments/assets/5e3d9027-ca0a-46cc-8764-ed82079dbd62)

- **WinFormsApp** — графический интерфейс

  ![вф](https://github.com/user-attachments/assets/beafe9ba-dd12-4786-9cdc-fcf3bc276bb8)
  
В WinForms приложение построено на одной главной форме. Все остальные части интерфейса реализованы с помощью UserControl, которые подгружаются, управляются и сменяются внутри этой формы.

## 🚀 Возможности

- 📝 Регистрация нового пользователя с валидацией
![Регистрация](https://github.com/user-attachments/assets/859d304e-472c-4868-b0e1-41dfb3308eb5)
![Регистрация консоль](https://github.com/user-attachments/assets/d8d446d9-604e-4339-a940-5f3e0b0861c1)
Валидация и нормализация имени при регистрации: разрешены только буквы русского алфавита, пробел, тире и апостроф. Далее программа обрабатывает пользовательский ввод, заменяя заглавные буквы на строчные, убирает дублирующиеся знаки, оставляя самый первый.
Также предусмотрена возможность не вводить отчество.

- ☰ Наличие главного меню
![гм](https://github.com/user-attachments/assets/e9e590f3-99fe-48c1-bb2f-528971afc6a3)
![гм консоль](https://github.com/user-attachments/assets/ccbf4e3e-eebf-4a30-83f9-0ec31e55ef23)

- 🎮 Прохождение викторины с ограничением по времени (15 секунд на вопрос)
![игра](https://github.com/user-attachments/assets/23c43c5a-7aed-4e15-9f14-eaf34ad6dfa5)
![игра консоль](https://github.com/user-attachments/assets/87c3498c-dc1a-4626-9f05-f338b1354447)
В квизе «Гений идиот» нужно быстро давать числовые ответы. Валидация ввода пропускает только цифры. Есть ограничение по времени и наглядная визуализация оставшегося времени.
В консольном приложении визуализация таймера реализована через сохранение текущей позиции курсора.

- 🧾 Сохранение результатов в JSON
<img width="1126" height="672" alt="image" src="https://github.com/user-attachments/assets/34e5d257-8120-4851-9b36-b5837c6b4ad5" />
Результаты сохраняются в JSON. При запуске программа читает историю из файла и отображает предыдущие попытки.

- 📊 Просмотр истории игр
![таблица](https://github.com/user-attachments/assets/548196b4-14b4-40df-b71a-fe29b0691b4c)
![таблица консоль](https://github.com/user-attachments/assets/23130abc-3aa3-4be6-94de-30d8103719d1)
  
- ➕ Добавление новых вопросов
![добавление вопросов](https://github.com/user-attachments/assets/9800908d-8008-46da-9dc0-82af0fb83eb9)
![добавление вопросов консоль](https://github.com/user-attachments/assets/4ca779bb-2254-442c-bbad-c0840b918c6f)
Добавление вопросов тоже идёт через сохранение в JSON-формат. Вопросы читаются из JSON-файла, и новые вопросы добавляются туда же

- ❌ Удаление пользовательских вопросов
![удаление](https://github.com/user-attachments/assets/f7aaebec-7216-4cae-9e24-5767712f38ed)
![удаление консоль](https://github.com/user-attachments/assets/73325e8d-ba40-42af-bc64-803b20f55d27)
Удаление вопросов организовано так, что удаляются только пользовательские вопросы. При их отсутствии операция недоступна.

## ⚙️ Технологии

- C#
- .NET
- WinForms
- JSON (Newtonsoft.Json)

## 💡 Особенности реализации

- Реализован таймер с обновлением строки в консоли
- Валидация пользовательского ввода (ФИО, вопросы)
- Нормализация текста
- Рандомизация вопросов
- Разделение логики и UI

## 📂 Хранение данных

- Вопросы: `QuestionList.json`
- История игр: `QuizHistory.json`
